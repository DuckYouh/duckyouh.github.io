Linux parancsok

-fájlok

  • ls: Listázás
  • nano {fájlnév}: Szövegszerkesztő megnyitása a megadott fájlnévvel
  • cat {fájlnév}: A megadott fájl tartalmának kiírása a képernyőre

-mappák

  • mkdir {mappanév}: Új mappa létrehozása a jelenlegi könyvtárban
  • cd {mappanév}: Belépés a megadott nevű mappába
  • cd ..: Visszalépés egy szinttel a mappa hierarchiában
  • rmdir {mappanév}: A megadott mappa törlése (csak akkor, ha üres)
  • pwd: A jelenlegi munkakönyvtár teljes elérési útvonala
  • tree: A jelenlegi könyvtárban található mappák és fájlok hierarchiáját ábrázoló fa-struktúra kiírása

-egyéb

  • echo: A megadott szöveg kiírása a képernyőre

Git parancsok

-github

  • git config --global user.name {felhasználónév}: regisztrálja a felhasználónevet
  • git config --global user.email {email cím}: regisztrálja az emailt
  • git clone {github link}: leklónozza a github mappát és tartalmát

-fájlok és mappák

  • git init {mappanév}: Létrehozza a gitet
  • git add {fájlnév}: Hozzáadja a megadott fájlt a git előkészítő részére
  • git add *: Hozzáad minden fájlt a git előkészítő részére
  • git commit -m “{Üzenet}”: Feltölti a fájlokat egy üzenettel az előkészítőből a gitre
  • git log: Listázza a verziókat

-elágazások

  • git branch: Listázza az elágazásokat
  • git branch {elágazásnév}: Létrehoz egy elágazást
  • git branch -d {elágazásnév}: Törli az elágazást
  • git checkout {elágazásnév}: Belép az elágazásba
  • git diff: Megadja a különbséget két verzió között
