Fájl megnyitása


file = open("teszt.txt","művelet",encoding="UTF-8")
Művelet típusok
  • "r" - Beolvasás
  • "a" - Bővítés
  • "w" - Írás
  • "x" - Létrehozás

  • "t" - Szövegtípus
  • "b" - Számtípus
